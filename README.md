# 2026 FIFA World Cup Predictor

An end-to-end machine learning project that predicts international football match outcomes and simulates the **2026 FIFA World Cup** using **historical match data, Elo ratings, recent team form, and Monte Carlo tournament simulation**.

---

## Overview

This project builds a forecasting pipeline for the 2026 FIFA World Cup by combining:

- **historical international match results**
- **pre-match Elo ratings**
- **rolling recent-form features**
- a **multi-class match prediction model**
- a **Monte Carlo tournament simulator**
- a **Streamlit web app** for interactive predictions and tournament simulations

The final system can:

- predict **win / draw / loss probabilities** for any matchup
- simulate **group-stage outcomes**
- estimate **knockout qualification odds**
- simulate the **full tournament** thousands of times to estimate **champion probabilities**
